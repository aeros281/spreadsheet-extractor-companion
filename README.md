# Spreadsheet Extractor Companion

A Chrome extension (Manifest V3) that extracts data from Google Sheets and generates CLI commands for the [spreadsheet-extractor](https://github.com/aeros281/spreadsheet-extractor) tool.

## Usage

1. Open a Google Sheets spreadsheet
2. Select a range of cells
3. Click the extension icon to open the popup
4. Configure column headers and options
5. Copy the generated `spreadsheet-extractor` CLI command

## Requirements

This extension generates commands for the CLI tool at [https://github.com/aeros281/spreadsheet-extractor](https://github.com/aeros281/spreadsheet-extractor). Install that tool to run the generated commands.

## Installation

Load the extension as an unpacked extension in Chrome:

1. Go to `chrome://extensions`
2. Enable **Developer mode**
3. Click **Load unpacked** and select this directory
